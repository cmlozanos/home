Base algorítmica

  • Herbert Kociemba describe el algoritmo de dos fases: primero reduce el cubo al subgrupo <U,D,R2,L2,F2,B2> y después resuelve dentro de ese subgrupo. kociemba.org
  • cube.js modela cubos 3x3 y porta ese algoritmo para resolver estados en navegadores modernos. GitHub
  • La cadena interna sigue el orden de caras U R F D L B documentado por Cube.fromString(). documentación
